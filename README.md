Team Name: Molten Minds (Team 17)

Theme interpretation: You are a blacksmith and must make weapons for your customers Core mechanics: Material Collection, Crafting Weapons, Serving customers on time

Controls:
- WASD/Arrow Keys to move

- Space to jump

- Shift to dash

- E to grab items,

- C to craft 

Known Issues and Bugs: Missing animations and lacking assets, Sprites clipping, Walls can be sticky, player can get softlocked when standing on items. UI improvements. 

Used Software: Unity
